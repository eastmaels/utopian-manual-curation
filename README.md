## Deploying

Install the [Heroku Toolbelt](https://toolbelt.heroku.com/).

```sh
$ git clone git@github.com:eastmaels/utopian-manual-curation.git # or clone your own fork
$ cd utopian-manual-curation
$ heroku create
$ git push heroku master
$ heroku open
```

or
